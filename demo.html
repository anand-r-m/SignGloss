What You're Seeing

This demo shows the complete SignGloss pipeline running in real-time inside the browser:

  1. Camera Capture — WebRTC captures the video feed from the user's webcam at 30 FPS.
  2. Landmark Extraction — MediaPipe detects 26 key landmarks (hands + body) and draws them as an overlay on the video.
  3. Feature Engineering — Position and velocity features (156 total) are computed per frame inside a Web Worker.
  4. Inference — A sliding window of 64 frames is fed to the ONNX model, which outputs probability distributions over 601 gloss classes.
  5. CTC Decoding — Greedy CTC decode collapses the output into readable ISL gloss tokens displayed in the results panel.

Everything runs client-side — no server, no backend, no data leaves your browser.
